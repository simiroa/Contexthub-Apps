# ComfyUI Creative Studio (Z)

## 소개
Z-Turbo 전용의 빠른 생성 워크스페이스입니다. 공용 Qt shell 위에서 빠른 프롬프트, 샘플링, 출력 옵션만 압축 노출합니다.

## 주요 기능
- 한 줄 헤더와 압축된 빠른 설정
- 좌측 결과 프리뷰와 최근 파일 목록
- 우측 `Prompt / Quick Settings / Sampling / Advanced` 접이식 섹션
- 요약형 `Export And Run` 카드
- 창 상태 복원과 우하단 리사이즈 그립

## 사용법
1. **Creative Studio (Z)** 실행
2. 프롬프트와 해상도, 스텝, CFG, 배치 수를 조정
3. 필요하면 출력 옵션을 펼쳐 수정
4. **Export** 또는 **Run** 실행
5. 좌측에서 최근 산출물과 프리뷰를 확인

## 요구사항
- **ComfyUI** 설치 필요
- **PySide6** 설치 필요
- Z-Turbo 실제 실행은 후속 런타임 연결 대상이며 현재는 빠른 셸/세션 export 중심
