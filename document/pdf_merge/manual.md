# PDF Merge (PDF 병합)

## 소개
여러 PDF를 하나의 파일로 병합합니다. Qt 기반 단일 컬럼 UI에서 순서를 위아래로 조정한 뒤 바로 병합할 수 있습니다.

## 사용법

1. 병합할 PDF를 선택하거나 앱 안에서 **Add PDFs** 로 추가합니다.
2. 리스트에서 파일을 선택합니다.
3. **Move Up / Move Down** 으로 병합 순서를 조정합니다.
4. 하단 실행 패널에서 출력 폴더와 파일 이름을 확인합니다.
5. **Merge PDFs** 를 눌러 병합합니다.

## 기능
- **순서 조정**: 파일 순서를 위/아래 버튼으로 재배치
- **선택 요약**: 선택한 PDF의 파일명, 경로, 크기 표시
- **출력 제어**: 출력 폴더와 파일 접두어 지정

## 출력
- 기본 파일명: `merged.pdf`
- 같은 이름이 이미 있으면 `_2`, `_3` 접미사를 붙여 새 파일 생성

## 참고
- v1 미리보기는 첫 페이지 썸네일이 아니라 파일 요약 정보 중심입니다.
