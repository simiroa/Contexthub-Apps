# PDF Split (PDF 분할)

## 소개
다중 페이지 PDF 문서를 페이지 단위로 분리합니다.

이 앱은 짧은 확인 UI 후 콘솔 흐름으로 처리됩니다.

## 사용법

1. PDF 파일 선택 후 우클릭
2. **Document → PDF Split** 선택
3. 확인 UI에서 출력 포맷 선택:
   - **PDF**
   - **PNG**
   - **JPEG**
4. **Split** 실행
5. 콘솔 진행 로그 확인

## 출력 파일
```
원본파일_split\원본파일_page_001.pdf
원본파일_split\원본파일_page_002.pdf
원본파일_split\원본파일_page_003.pdf
...
```

## 활용 예시
- 계약서에서 서명 페이지만 추출
- 책 한 챕터만 별도 저장
- 스캔 문서 개별 관리
