# Simple Normal Roughness

이미지로부터 Normal Map과 Roughness Map을 생성하는 도구입니다.

## 주요 기능

- **Strip Check (모드 선택)**: 상단의 [Normal], [Roughness], [Both] 버튼을 통해 생성할 맵의 종류를 선택합니다.
- **실시간 프리뷰**: 드래그 앤 드롭으로 이미지를 추가하면 선택한 모드에 따른 결과물이 즉시 프리뷰됩니다.
- **동적 파라미터**: 선택한 모드에 필요한 설정값(강도, 대비 등)만 화면에 표시됩니다.
- **일괄 처리**: 이미지를 드랍하고 `Export Maps` 버튼을 누르면 설정된 값으로 결과물이 자동 저장됩니다.

## 사용 방법

1. 앱을 실행합니다.
2. 소스 이미지를 프리뷰 영역으로 드래그 앤 드롭합니다.
3. 상단 스트립에서 원하는 모드([Normal] / [Roughness] / [Both])를 선택합니다.
4. 하단 파라미터를 조절하여 결과를 확인합니다.
5. `Export Maps` 버튼을 눌러 저장합니다.
6. `Open Output` 버튼으로 결과 폴더를 확인합니다.
