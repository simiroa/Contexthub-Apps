# Merge to EXR (EXR 병합)

## 소개
여러 이미지를 하나의 멀티레이어 EXR 파일로 병합합니다. Qt 기반 단일 컬럼 UI에서 레이어 이름, 모드, 활성 상태를 직접 수정하고 순서를 조정할 수 있습니다.

## 사용법

1. 병합할 이미지들을 선택하거나 앱 안에서 **Add Images** 로 추가합니다.
2. 자동 제안된 레이어 리스트를 확인합니다.
3. 선택한 행의 레이어 이름, 모드(`RGB`, `RGBA`, `L`, `R`, `G`, `B`, `A`), `Enabled`, `Invert`, `Linearize` 값을 수정합니다.
4. **Move Up / Move Down** 으로 레이어 순서를 정리합니다.
5. 하단 실행 패널에서 출력 폴더와 파일 이름을 확인한 뒤 **Export EXR** 를 실행합니다.

## 기능
- **자동 제안**: 파일명 기반 레이어 이름/모드 자동 설정
- **수동 수정**: 각 레이어 행의 이름, 모드, 토글 값 직접 편집
- **순서 조정**: 레이어 리스트를 위/아래 버튼으로 재배치
- **의존성 프리체크**: `cv2`, `imageio`, `OpenEXR`, `Imath` 누락 시 실행 전에 안내

## 지원 입력 포맷
- PNG, JPG, TIFF, BMP, TGA, EXR

## 출력 파일
- 기본 파일명: `MultiLayer_Output.exr`
- 같은 이름이 이미 있으면 `_2`, `_3` 접미사를 붙여 새 파일 생성

## 활용 예시
- 렌더 패스를 하나의 파일로 관리
- Nuke/AfterEffects 합성 준비
- 렌더팜 출력 정리

## 의존성
- `cv2`
- `imageio`
- `OpenEXR`
