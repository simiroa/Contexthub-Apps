# Interpolate 30fps (프레임 보간)

## 소개
FFmpeg Minterpolate 필터를 사용하여 저프레임 영상(24fps 이하)을 30fps로 부드럽게 보간합니다.

이 앱은 짧은 확인 UI 후 콘솔 진행 로그로 처리됩니다.

## 사용법

1. 비디오 파일 선택 후 우클릭
2. **Video → Interpolate 30fps** 선택
3. 파일 개수 확인 후 **Interpolate** 실행
4. 콘솔 진행 로그 확인
5. 결과물 확인

## 출력 파일
- `원본파일명_30fps.mp4`

## 기술 세부사항
- **방식**: FFmpeg minterpolate 필터
- **출력 FPS**: 30fps 고정
- **품질**: 원본 유지 (인코딩은 새로 수행)

## 활용 예시
- 타임랩스 영상 부드럽게
- 24fps 영화를 30fps로
- 저프레임 게임 영상 개선

## 참고
- AI 보간이 아닌 알고리즘 기반 보간
- 빠른 움직임에서 아티팩트 발생 가능
- 더 나은 품질이 필요하면 AI 보간 도구 사용 권장
