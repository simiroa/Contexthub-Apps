# Remove Audio (오디오 제거)

## 소개
비디오에서 오디오 트랙을 제거하여 무음 영상을 생성합니다.

이 앱은 무거운 GUI 없이 콘솔 흐름으로 바로 처리됩니다.

## 사용법

1. 비디오 파일 선택 후 우클릭
2. **Video → Remove Audio** 선택
3. 콘솔 진행 로그 확인
4. 원본과 같은 폴더에 무음 영상 생성

## 출력 파일
- `원본파일명_mute.원본확장자`

## 특징
- 원본 영상 스트림 유지 (`-c:v copy`)
- 빠른 처리 속도
- 원본 파일 보존
- 여러 파일 배치 처리 가능

## 활용 예시
- 저작권 음악 제거
- 배경 소음 제거
- 자체 배경음악 삽입 준비
- GIF 변환 전 오디오 제거

## 의존성
- FFmpeg (허브 환경에서 제공 또는 경로 설정 필요)
