# Extract Audio (오디오 추출)

## 소개
비디오 파일에서 오디오 트랙만 추출하여 MP3 파일로 저장합니다.

이 앱은 무거운 편집 GUI 대신, 짧은 확인 UI 후 콘솔 흐름으로 바로 처리됩니다.

## 사용법

1. 비디오 파일 선택 후 우클릭
2. **Video → Extract Audio** 선택
3. 파일 개수 확인 후 **Extract** 실행
4. 콘솔 진행 로그 확인

## 출력 파일
- `원본파일명.mp3`
- 원본 비디오와 같은 폴더에 저장

## 활용 예시
- 영상에서 배경음악 추출
- 강의 영상의 음성만 저장
- 음악 비디오에서 음원 추출
- 팟캐스트 음성 저장

## 의존성
- FFmpeg (허브 환경에서 제공 또는 경로 설정 필요)
