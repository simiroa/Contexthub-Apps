# Marigold PBR Generator

Marigold AI를 사용하여 단일 이미지에서 고품질의 PBR(Physically Based Rendering) 텍스처 맵을 생성합니다.

## 주요 기능

- **지오메트리 추출**: 깊이(Depth) 및 법선(Normal) 맵 생성.
- **재질 분해**: 알베도(Albedo), 거칠기(Roughness), 금속성(Metallicity) 맵 추출.
- **ORM 패킹**: Ambient Occlusion, Roughness, Metallic 데이터를 하나의 파일로 패킹.
- **표준 레이아웃**: 입력 목록과 미리보기가 결합된 표준 인터페이스를 제공합니다.

## 사용 방법

1. **이미지 추가**: 좌측 하단의 "＋" 버튼을 누르거나 이미지 파일을 드래그 앤 드롭합니다.
2. **미리보기**: 목록에서 이미지를 선택하면 상단 프리뷰 화면에 표시됩니다.
3. **설정 조절**: 우측 패널에서 생성하고자 하는 맵 종류와 품질 옵션(Speed, Balanced, Quality)을 선택합니다.
4. **실행**: "Run" 버튼을 눌러 생성을 시작합니다.
5. **결과 확인**: 생성된 맵은 원본 이미지와 같은 폴더에 저장됩니다.

## 주의 사항

- 이 도구는 상당한 양의 VRAM을 소모합니다. VRAM이 부족한 경우 해상도를 낮추거나 FP16 옵션을 활성화하세요.
- 최초 실행 시 AI 모델 다운로드로 인해 시간이 다소 소요될 수 있습니다.
