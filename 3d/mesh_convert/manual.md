# Mesh Convert (3D 메쉬 변환)

## 소개
FBX, OBJ, GLB, USD 등 다양한 3D 메쉬 포맷을 상호 변환합니다.

## 지원 포맷

| 포맷 | 입력 | 출력 | 용도 |
|------|-----|-----|------|
| **FBX** | ✓ | ✓ | Autodesk, 게임 엔진 |
| **OBJ** | ✓ | ✓ | 범용, 호환성 최고 |
| **GLB/GLTF** | ✓ | ✓ | 웹/AR/VR |
| **USD/USDC** | ✓ | ✓ | 영화/VFX, Pixar |
| **STL** | ✓ | ✓ | 3D 프린팅 |
| **PLY** | ✓ | ✓ | 스캔 데이터 |

## 사용법

1. 3D 파일 선택 후 우클릭
2. **3D → Mesh Convert** 선택
3. Qt 창에서 입력 파일 목록과 선택 상태 확인
4. 출력 포맷과 저장 위치를 선택
5. 변환 실행 후 결과 폴더 확인

## 옵션
- **출력 포맷**: OBJ, FBX, GLB, USD, PLY, STL
- **하위 폴더 저장**: `Converted_Mesh` 폴더에 결과 저장
