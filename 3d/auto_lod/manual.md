# Auto LOD (자동 LOD 생성)

## 소개
고해상도 3D 메쉬의 폴리곤 수를 자동으로 줄여 게임/실시간 렌더링에 적합한 LOD(Level of Detail) 모델을 생성합니다.

## LOD란?
- 거리에 따라 다른 디테일 레벨의 모델 사용
- 가까이 있으면 고해상도, 멀리 있으면 저해상도
- 렌더링 성능 최적화

## 사용법

1. 3D 메쉬 파일 선택 후 우클릭
2. **3D → Auto LOD** 선택
3. full Qt 창에서 단일 입력 메쉬와 3D 프리뷰 확인
4. 좌측 3D Preview 카드에서 모델을 회전/줌하고, 필요하면 다른 파일을 드래그해서 교체
5. LOD 옵션 설정:
   - **LOD 레벨**: 생성할 LOD 개수
   - **감소 비율**: 각 레벨별 폴리곤 감소율
   - **보존 옵션**: UV, 노멀, 경계 유지
6. 생성 실행

## 프리뷰 조작
- 좌클릭 드래그: 회전
- 휠: 줌
- `Fit`: 기본 프레이밍
- `Reset`: 기본 시점 복귀
- `Wireframe`: 와이어프레임/쉐이디드 토글

## 출력 경로
- 현재 UI는 입력 파일 옆에 `모델명_lod` 폴더를 기준 출력 위치로 사용합니다.
- LOD 이름은 `모델명_lod_50.obj` 같은 규칙으로 계산됩니다.
- 실제 단순화 파이프라인은 아직 placeholder 상태이므로, 현재 단계에서는 결과 경로 계약과 UI 흐름이 먼저 정리되어 있습니다.
