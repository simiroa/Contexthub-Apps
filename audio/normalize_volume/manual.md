# Normalize Volume (볼륨 정규화)

## 소개
여러 오디오 파일의 볼륨을 일정한 수준으로 평준화하여 소리 크기 차이를 없앱니다.

이 앱은 짧은 확인 UI 후 콘솔 흐름으로 처리됩니다.

## 사용법

1. 오디오 파일(들) 선택 후 우클릭
2. **Audio → Normalize Volume** 선택
3. 파일 개수 확인 후 **Normalize** 실행
4. 콘솔 진행 로그 확인

## 출력 파일
- `원본파일명_normalized.원본확장자`

## 정규화 방식
- FFmpeg `loudnorm` 필터 사용
- 기본 목표값은 앱 기본 설정을 따름

## 활용 예시
- 팟캐스트 에피소드 볼륨 통일
- 음악 믹스 수준 맞추기
- 다양한 소스 오디오 편집
- YouTube 업로드 전 음량 조절

## 의존성
- FFmpeg (자동 설치)
