# Compare Images (이미지 비교)

## 소개
두 이미지를 나란히 놓고 시각적으로 비교하거나 차이점을 하이라이트합니다.

## 주요 기능
- **Side-by-Side 비교**: 두 이미지 나란히 표시
- **오버레이 비교**: 투명도 조절로 겹쳐서 비교
- **차이점 강조**: 픽셀 차이 하이라이트
- **줌/패닝**: 세부 영역 확대 비교
- **메타데이터 비교**: 해상도, 포맷, 크기 비교

## 사용법

1. 비교할 이미지 2개 선택 (Ctrl+클릭)
2. 우클릭 → **Image → Compare Images** 선택
3. 비교 모드 선택:
   - **Split View**: 좌우 분할
   - **Overlay**: 투명도 슬라이더로 비교
   - **Diff**: 차이점만 표시
4. 줌/패닝으로 세부 비교

## 활용 예시
- 편집 전후 비교
- 압축 품질 확인
- 버전 간 차이 확인
- 리터칭 결과 확인

## 지원 포맷
- PNG, JPG, JPEG, WebP, BMP, TIFF
