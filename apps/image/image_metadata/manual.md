# Image Metadata Tool (이미지 메타데이터)

## 소개
이미지 파일의 EXIF, IPTC 등 메타데이터를 조회하고 편집합니다.

## 주요 기능
- **메타데이터 조회**: EXIF, IPTC, XMP 정보 표시
- **메타데이터 편집**: 날짜, 작성자, 제목 등 수정
- **메타데이터 삭제**: 개인정보 보호용 스트립
- **배치 처리**: 여러 이미지 일괄 편집

## 조회 가능한 정보

| 카테고리 | 정보 |
|---------|-----|
| **촬영 정보** | 카메라, 렌즈, 셔터속도, ISO |
| **위치 정보** | GPS 좌표, 촬영 장소 |
| **시간 정보** | 촬영일, 수정일 |
| **저작권** | 작성자, 저작권, 설명 |

## 사용법

1. 이미지 파일 선택 후 우클릭
2. **Image → Image Metadata Tool** 선택
3. 메타데이터 조회/편집:
   - **View**: 모든 메타데이터 확인
   - **Edit**: 특정 항목 수정
   - **Strip**: 모든 메타데이터 삭제
4. 변경 사항 저장

## 활용 예시
- 촬영 정보 확인
- SNS 업로드 전 위치정보 삭제
- 저작권 정보 추가
- 날짜 정보 수정
