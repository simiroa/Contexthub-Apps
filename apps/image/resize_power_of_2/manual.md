# Resize Power of 2 (2의 거듭제곱 리사이즈)

## 소개
이미지를 256, 512, 1024, 2048 등 2의 거듭제곱 크기로 리사이즈합니다. 게임 엔진 텍스처 최적화에 필수입니다.

## 왜 2의 거듭제곱인가?
- GPU 메모리 최적화
- 밉맵 자동 생성 지원
- 게임 엔진 호환성 (Unity, Unreal)
- 텍스처 압축 효율

## 사용법

1. 이미지 파일(들) 선택 후 우클릭
2. **Image → Resize Power of 2** 선택
3. 리사이즈 모드 선택:
   - **Nearest**: 가장 가까운 크기로
   - **Up**: 더 큰 크기로
   - **Down**: 더 작은 크기로
   - **Custom**: 특정 크기 선택
4. 실행

## 지원 크기
| 크기 | 용도 |
|-----|-----|
| 256×256 | UI 아이콘, 소형 텍스처 |
| 512×512 | 일반 텍스처 |
| 1024×1024 | 중형 텍스처 |
| 2048×2048 | 대형/히어로 텍스처 |
| 4096×4096 | 초고해상도 텍스처 |
