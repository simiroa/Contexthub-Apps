# PDF Merge (PDF 병합)

## 소개
여러 개의 PDF 문서를 하나의 연속된 PDF 파일로 합칩니다.

## 사용법

1. 합치고 싶은 PDF 파일들을 모두 선택 (Ctrl+클릭)
2. 우클릭 → **Document → PDF Merge** 선택
3. 파일 순서 확인 및 조정 (드래그로 순서 변경)
4. **Merge** 버튼으로 병합 실행
5. 결과 파일 저장 위치 선택

## 기능
- **순서 조정**: 병합 전 페이지 순서 드래그 조정
- **미리보기**: 각 PDF 첫 페이지 썸네일 표시
- **배치 처리**: 수십 개 파일 동시 병합 가능

## 출력
- `merged.pdf` 또는 지정한 이름으로 저장
- 원본 파일은 그대로 유지
