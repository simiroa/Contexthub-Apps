# PDF Split (PDF 분할)

## 소개
다중 페이지 PDF 문서를 개별 페이지 파일로 분리하거나 특정 페이지 범위를 추출합니다.

## 사용법

1. PDF 파일 선택 후 우클릭
2. **Document → PDF Split** 선택
3. 분할 옵션 선택:
   - **전체 분할**: 모든 페이지를 개별 파일로
   - **범위 지정**: 특정 페이지만 추출 (예: 1-5, 10-15)
   - **짝수/홀수**: 짝수 또는 홀수 페이지만
4. **Split** 버튼으로 실행

## 출력 파일
```
원본파일_page_001.pdf
원본파일_page_002.pdf
원본파일_page_003.pdf
...
```

## 활용 예시
- 계약서에서 서명 페이지만 추출
- 책 한 챕터만 별도 저장
- 스캔 문서 개별 관리
