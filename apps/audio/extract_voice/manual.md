# Extract Voice (보컬 추출)

## 소개
Demucs AI를 사용하여 오디오에서 배경음악을 제거하고 목소리(보컬)만 선명하게 추출합니다.

## 사용법

1. 오디오/영상 파일 선택 후 우클릭
2. **Audio → Extract Voice** 선택
3. AI 처리 완료 대기 (곡당 2-5분)
4. 보컬 파일 생성

## 출력 파일
- `원본파일명_vocals.wav`

## 기술 세부사항
- Demucs 모델 사용
- 악기 트랙 제거
- 보컬만 추출

## 활용 예시
- 아카펠라 트랙 제작
- 보컬 샘플링
- 대사/내레이션 추출
- 음성 인식 전처리

## 시스템 요구사항
- NVIDIA GPU 권장
- VRAM 4GB 이상
- CPU 모드도 지원 (느림)

## 의존성
- Demucs
- PyTorch
