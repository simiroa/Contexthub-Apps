# Extract BGM (배경음악 추출)

## 소개
Demucs AI를 사용하여 오디오에서 보컬을 제거하고 배경음악(BGM)만 추출합니다.

## 사용법

1. 오디오/영상 파일 선택 후 우클릭
2. **Audio → Extract BGM** 선택
3. AI 처리 완료 대기 (곡당 2-5분)
4. BGM 파일 생성

## 출력 파일
- `원본파일명_bgm.wav`

## 기술 세부사항
- Demucs 모델 사용
- 보컬 트랙만 제거
- 드럼, 베이스, 기타 등 유지

## 활용 예시
- 노래방 반주 제작
- 영상 배경음악 재사용
- 리믹스/매시업 작업
- 커버 곡 반주 추출

## 시스템 요구사항
- NVIDIA GPU 권장
- VRAM 4GB 이상
- CPU 모드도 지원 (느림)

## 의존성
- Demucs
- PyTorch
