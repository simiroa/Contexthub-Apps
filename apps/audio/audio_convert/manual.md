# Audio Convert (오디오 변환)

## 소개
오디오 파일의 포맷을 변환합니다. MP3, WAV, FLAC, OGG, M4A 등 다양한 형식을 지원합니다.

## 지원 포맷

| 포맷 | 입력 | 출력 | 특징 |
|------|-----|-----|------|
| **MP3** | ✓ | ✓ | 범용, 손실 압축 |
| **WAV** | ✓ | ✓ | 무손실, 대용량 |
| **FLAC** | ✓ | ✓ | 무손실 압축 |
| **OGG** | ✓ | ✓ | 오픈소스, 웹 호환 |
| **M4A/AAC** | ✓ | ✓ | Apple, 고효율 |
| **AIFF** | ✓ | ✓ | Apple 무손실 |

## 사용법

1. 오디오 파일(들) 선택 후 우클릭
2. **Audio → Audio Convert** 선택
3. 옵션 설정:
   - **출력 포맷**: MP3, WAV, FLAC 등
   - **비트레이트**: 128kbps ~ 320kbps
   - **샘플레이트**: 44.1kHz, 48kHz 등
4. 변환 시작

## 배치 처리
- 여러 파일 동시 변환 가능
- 폴더 단위 변환 지원

## 의존성
- FFmpeg (자동 설치)
