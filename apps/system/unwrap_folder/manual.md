# Unwrap Folder (폴더 펼치기)

## 소개
폴더 안의 모든 파일을 부모 폴더로 꺼내오고, 빈 폴더를 삭제합니다. 불필요한 폴더 구조를 해제합니다.

## 사용법

1. 펼칠 폴더 선택 후 우클릭
2. **System → Unwrap Folder** 선택
3. 확인 대화상자에서 승인
4. 파일이 부모 폴더로 이동, 빈 폴더 삭제

## 동작 예시
```
Before:
└── Projects/
    └── Download_2024/
        └── Assets/
            ├── image1.png
            └── image2.png

After:
└── Projects/
    ├── image1.png
    └── image2.png
```

## 활용 예시
- 압축 해제 후 불필요한 중첩 폴더 정리
- 복잡한 폴더 구조 평탄화
- 파일 정리 자동화

## 주의사항
- 동일 파일명 충돌 시 경고
- 원복 불가 (휴지통 없음)
- 중요 폴더는 백업 권장
