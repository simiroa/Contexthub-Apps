# Batch Rename (일괄 이름 변경)

## 소개
수백 개의 파일명을 일정한 규칙(넘버링, 날짜, 문자열 치환, 패턴)으로 빠르게 변경합니다.

## 주요 기능

### 이름 변경 규칙
| 규칙 | 예시 | 설명 |
|------|-----|------|
| **순번** | img_001, img_002 | 자동 넘버링 |
| **날짜** | 20260112_photo | 현재/파일 날짜 삽입 |
| **치환** | old→new | 특정 문자열 교체 |
| **접두어/접미어** | prefix_name_suffix | 앞뒤 문자 추가 |
| **대소문자** | NAME→name | 대소문자 변환 |
| **정규식** | 패턴 매칭 치환 | 고급 패턴 적용 |

## 사용법

1. 이름을 변경할 파일들 선택 후 우클릭
2. **System → Batch Rename** 선택
3. 규칙 설정:
   - 패턴 입력
   - 시작 번호 설정
   - 자릿수 지정
4. **미리보기**로 결과 확인
5. **적용** 버튼으로 일괄 적용

## 미리보기 기능
- 변경 전/후 파일명 비교
- 충돌(중복) 자동 감지
- 실행 전 검토 가능

## 팁
- 실수 방지를 위해 항상 미리보기 확인
- 중요 파일은 백업 후 진행 권장
