# Move to New Folder (새 폴더로 이동)

## 소개
선택한 파일들을 담을 새 폴더를 즉시 생성하고 한 번에 이동시킵니다.

## 사용법

1. 정리할 파일들 선택 (Ctrl+클릭으로 다중 선택)
2. 우클릭 → **System → Move to New Folder** 선택
3. 폴더 이름 입력
4. 자동으로 폴더 생성 및 파일 이동

## 기능
- **즉시 생성**: 폴더 생성과 이동을 한 번에
- **이름 제안**: 선택된 파일 기반 폴더명 자동 제안
- **안전 이동**: 동일 파일명 충돌 시 경고

## 활용 예시
- 다운로드 파일 정리
- 프로젝트별 폴더 생성
- 날짜별 파일 분류
- 작업 파일 그룹화

## 팁
- 여러 타입의 파일을 한번에 선택 가능
- 이동 후 원래 위치에 파일이 없어지므로 주의
