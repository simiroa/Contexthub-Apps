# Clean Empty Folders (빈 폴더 정리)

## 소개
프로젝트나 폴더 내에 불필요하게 남아있는 빈 폴더들을 재귀적으로 찾아서 일괄 삭제합니다.

## 사용법

1. 정리할 상위 폴더 선택 후 우클릭
2. **System → Clean Empty Folders** 선택
3. 빈 폴더 스캔 결과 확인
4. **Delete** 버튼으로 일괄 삭제

## 기능
- **재귀 스캔**: 모든 하위 폴더까지 검색
- **미리보기**: 삭제 전 폴더 목록 확인
- **안전 삭제**: 휴지통 이동 옵션 지원

## 활용 예시
- 파일 정리 후 남은 빈 폴더 제거
- 프로젝트 구조 정리
- 다운로드 폴더 청소
- SVN/Git 정리 후 잔여 폴더 제거

## 주의사항
- 숨김 폴더도 검사됩니다
- 삭제 전 미리보기 확인 권장
