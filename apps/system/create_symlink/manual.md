# Create Symlink (심볼릭 링크 생성)

## 소개
파일이나 폴더에 대한 심볼릭 링크(바로가기와 유사)를 생성합니다. 원본 파일을 복사하지 않고 참조합니다.

## 심볼릭 링크란?
- 원본 파일/폴더를 가리키는 참조
- 바로가기와 비슷하지만 시스템 레벨에서 동작
- 용량을 차지하지 않음

## 사용법

1. 파일 또는 폴더 선택 후 우클릭
2. **System → Create Symlink** 선택
3. 링크 생성 위치 선택
4. 링크 이름 입력 (선택)
5. 생성 완료

## 활용 예시
- 동일 파일을 여러 위치에서 사용
- 프로젝트 폴더 간 에셋 공유
- SSD와 HDD 간 폴더 연결
- 게임 모드 폴더 관리

## 주의사항
- 관리자 권한 필요 (Windows)
- 원본 삭제 시 링크 무효화
- 네트워크 드라이브에는 제한적 지원

## 링크 유형
- **심볼릭 링크**: 파일/폴더 모두 가능
- **하드 링크**: 파일만, 같은 드라이브
- **정션**: 폴더만, 같은 드라이브
