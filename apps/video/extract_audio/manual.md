# Extract Audio (오디오 추출)

## 소개
비디오 파일에서 오디오 트랙만 추출하여 MP3 또는 WAV 파일로 저장합니다.

## 사용법

1. 비디오 파일 선택 후 우클릭
2. **Video → Extract Audio** 선택
3. 출력 형식 선택:
   - **MP3**: 용량 최적화 (128kbps~320kbps)
   - **WAV**: 무손실 원본 품질
   - **AAC**: 고효율 압축
4. 추출 시작

## 출력 파일
- `원본파일명.mp3` 또는 `.wav`
- 원본 비디오와 같은 폴더에 저장

## 활용 예시
- 영상에서 배경음악 추출
- 강의 영상의 음성만 저장
- 음악 비디오에서 음원 추출
- 팟캐스트 음성 저장

## 의존성
- FFmpeg (자동 설치)
