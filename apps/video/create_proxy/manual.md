# Create Proxy (프록시 생성)

## 소개
고화질 영상의 저해상도/저용량 프록시 파일을 생성하여 편집 시 성능을 향상시킵니다.

## 사용법

1. 고화질 영상 파일 선택 후 우클릭
2. **Video → Create Proxy** 선택
3. 프록시 옵션 설정:
   - **해상도**: 1/2, 1/4 축소
   - **코덱**: H.264, ProRes Proxy
   - **비트레이트**: 저용량 설정
4. 프록시 생성

## 출력 파일
- `원본파일명_proxy.mp4`

## 활용 워크플로우
1. 프록시 파일로 편집 작업 수행
2. 편집 완료 후 원본으로 교체
3. 최종 렌더링

## 활용 예시
- 4K/8K 영상 편집
- 저사양 PC에서 편집
- 네트워크 편집 환경

## 지원 편집 프로그램
- Premiere Pro
- DaVinci Resolve
- Final Cut Pro

## 의존성
- FFmpeg (자동 설치)
