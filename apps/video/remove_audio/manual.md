# Remove Audio (오디오 제거)

## 소개
비디오에서 오디오 트랙을 완전히 제거하여 무음 영상을 생성합니다.

## 사용법

1. 비디오 파일 선택 후 우클릭
2. **Video → Remove Audio** 선택
3. 처리 완료 대기
4. 무음 영상 생성

## 출력 파일
- `원본파일명_mute.mp4`

## 특징
- 원본 영상 품질 유지 (재인코딩 최소화)
- 빠른 처리 속도
- 원본 파일 보존

## 활용 예시
- 저작권 음악 제거
- 배경 소음 제거
- 자체 배경음악 삽입 준비
- GIF 변환 전 오디오 제거

## 의존성
- FFmpeg (자동 설치)
