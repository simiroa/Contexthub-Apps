# Video Convert (비디오 변환)

## 소개
비디오 파일의 포맷, 코덱, 비트레이트를 변환합니다. FFmpeg 기반으로 고화질 변환을 지원합니다.

## 지원 포맷

| 포맷 | 코덱 | 용도 |
|------|-----|------|
| **MP4** | H.264/H.265 | 범용, 웹 호환 |
| **MOV** | ProRes/H.264 | Apple 생태계, 편집용 |
| **AVI** | MPEG-4 | 레거시 호환 |
| **MKV** | 다양한 코덱 | 고급 기능, 자막 내장 |
| **WebM** | VP9/AV1 | 웹 최적화 |
| **GIF** | - | 애니메이션 이미지 |

## 사용법

1. 변환할 비디오 파일 선택 후 우클릭
2. **Video → Video Convert** 클릭
3. 변환 옵션 설정:
   - **출력 포맷**: MP4, MOV, AVI, MKV, WebM, GIF
   - **비트레이트**: 품질/용량 조절
   - **해상도**: 원본 유지 또는 변경
   - **FPS**: 프레임레이트 조절
4. **Convert** 버튼으로 변환 시작

## 고급 옵션
- **CRF (Constant Rate Factor)**: 품질 기반 인코딩
- **프리셋**: ultrafast ~ veryslow (속도/품질 균형)
- **오디오 코덱**: AAC, MP3, 무음 제거
