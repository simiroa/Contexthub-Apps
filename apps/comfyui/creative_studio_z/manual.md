# ComfyUI Creative Studio (Z)

## 소개
Z-Image Turbo를 활용한 초고속 이미지 생성 워크스페이스입니다. 아이콘, 에셋, 빠른 컨셉 작업에 최적화되어 있습니다.

## 주요 기능
- **Z-Turbo 엔진**: 초고속 이미지 생성 (2-4초)
- **프롬프트 레이어**: 다중 프롬프트 조합
- **배치 생성**: 여러 변형 동시 생성
- **실시간 미리보기**: 프롬프트 변경 즉시 반영

## 사용법

1. 바탕화면 우클릭 또는 메뉴에서 **Creative Studio (Z)** 실행
2. 프롬프트 입력:
   - 메인 프롬프트 (원하는 이미지 설명)
   - 네거티브 프롬프트 (제외할 요소)
3. 생성 옵션 설정:
   - 해상도
   - 배치 개수
   - 시드 값
4. **Generate** 클릭

## 요구사항
- **ComfyUI**: 설치 및 경로 설정 필요
- **Z-Turbo 모델**: ComfyUI에 다운로드
- GPU VRAM 8GB 이상 권장

## 활용 예시
- 게임 아이콘 대량 생성
- UI 에셋 컨셉
- 빠른 아이디어 스케치
