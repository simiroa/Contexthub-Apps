# Open Web UI (ComfyUI 웹 열기)

## 소개
설치된 ComfyUI의 웹 대시보드를 직접 엽니다. 서버가 꺼져 있다면 자동으로 시작합니다.

## 사용법

1. **ComfyUI → Open Web UI** 선택
2. 자동 동작:
   - ComfyUI 서버 상태 확인
   - 서버가 꺼져 있으면 자동 시작
   - 웹 브라우저에서 UI 열기

## 기본 URL
- `http://127.0.0.1:8188`

## 기능
- **서버 자동 시작**: 수동 실행 불필요
- **브라우저 열기**: 기본 브라우저로 UI 접속
- **상태 확인**: 서버 실행 여부 감지

## 활용 예시
- ComfyUI 워크플로우 직접 편집
- 커스텀 노드 테스트
- 고급 파이프라인 구성

## 요구사항
- **ComfyUI**: 설치 완료
- Manager → Settings에서 경로 설정
