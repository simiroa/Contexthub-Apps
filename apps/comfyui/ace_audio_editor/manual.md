# Creative Audio Studio (ACE)

## 소개
ComfyUI와 연동하여 AI 기반 오디오 생성, 편집, 리페인팅 기능을 제공하는 통합 오디오 스튜디오입니다.

## 주요 기능
- **오디오 생성**: 텍스트 프롬프트로 음악/효과음 생성
- **음성 변조**: 목소리 변환 및 TTS
- **오디오 리페인팅**: 기존 오디오 스타일 변환
- **노이즈 제거**: AI 기반 잡음 제거
- **미디 변환**: 오디오를 미디 노트로

## 사용법

1. 오디오 파일 선택 후 우클릭 또는 직접 실행
2. **Audio → Creative Audio Studio (ACE)** 선택
3. 작업 유형 선택:
   - **Generate**: 새 오디오 생성
   - **Edit**: 기존 오디오 편집
   - **Convert**: 형식/스타일 변환
4. 프롬프트 또는 설정 입력
5. 처리 시작

## 요구사항
- **ComfyUI**: 설치 및 경로 설정 필요
- **ACE 노드**: ComfyUI에 ACE 노드 팩 설치
