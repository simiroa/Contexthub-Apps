# Auto LOD (자동 LOD 생성)

## 소개
고해상도 3D 메쉬의 폴리곤 수를 자동으로 줄여 게임/실시간 렌더링에 적합한 LOD(Level of Detail) 모델을 생성합니다.

## LOD란?
- 거리에 따라 다른 디테일 레벨의 모델 사용
- 가까이 있으면 고해상도, 멀리 있으면 저해상도
- 렌더링 성능 최적화

## 사용법

1. 3D 메쉬 파일 선택 후 우클릭
2. **3D → Auto LOD** 선택
3. LOD 옵션 설정:
   - **LOD 레벨**: 생성할 LOD 개수 (3-5 권장)
   - **감소 비율**: 각 레벨별 폴리곤 감소율
4. 생성 실행

## 출력 파일
```
모델명_LOD0.fbx  # 원본 (100%)
모델명_LOD1.fbx  # 50%
모델명_LOD2.fbx  # 25%
모델명_LOD3.fbx  # 10%
```

## 활용 예시
- 게임 에셋 최적화
- VR/AR 콘텐츠 제작
- 웹 3D 뷰어용 경량화
