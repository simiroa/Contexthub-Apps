# Mesh Convert (3D 메쉬 변환)

## 소개
FBX, OBJ, GLB, USD 등 다양한 3D 메쉬 포맷을 상호 변환합니다.

## 지원 포맷

| 포맷 | 입력 | 출력 | 용도 |
|------|-----|-----|------|
| **FBX** | ✓ | ✓ | Autodesk, 게임 엔진 |
| **OBJ** | ✓ | ✓ | 범용, 호환성 최고 |
| **GLB/GLTF** | ✓ | ✓ | 웹/AR/VR |
| **USD/USDC** | ✓ | ✓ | 영화/VFX, Pixar |
| **STL** | ✓ | ✓ | 3D 프린팅 |
| **PLY** | ✓ | ✓ | 스캔 데이터 |

## 사용법

1. 3D 파일 선택 후 우클릭
2. **3D → Mesh Convert** 선택
3. 출력 포맷 선택
4. 변환 옵션 설정 (스케일, 축 등)
5. 변환 실행

## 옵션
- **스케일 조정**: 단위 변환 (m, cm, inch)
- **축 변환**: Y-up ↔ Z-up
- **텍스처 포함**: 재질/텍스처 번들

## 의존성
- Blender (고급 변환)
- pymeshlab (기본 변환)
