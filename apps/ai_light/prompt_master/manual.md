# Prompt Master (프롬프트 마스터)

## 소개
AI에게 전달할 프롬프트를 체계적으로 관리하고, 최적의 형태로 다듬어주는 도구입니다.

## 주요 기능
- **프롬프트 저장**: 자주 쓰는 프롬프트 라이브러리
- **카테고리 관리**: 용도별 프롬프트 분류
- **프롬프트 최적화**: AI가 잘 이해하는 형식으로 변환
- **변수 치환**: 템플릿 변수 지원

## 사용법

1. 트레이 메뉴에서 **Prompt Master** 실행
2. 프롬프트 작성 또는 라이브러리에서 선택
3. 최적화 옵션:
   - **Expand**: 세부 설명 추가
   - **Simplify**: 간결하게 정리
   - **Translate**: 영어로 번역
4. 결과 복사하여 사용

## 템플릿 변수 예시
```
{subject}의 {style} 스타일 이미지, {quality} 품질
→ 고양이의 수채화 스타일 이미지, 고해상도 품질
```

## 활용 예시
- Stable Diffusion 프롬프트 관리
- ChatGPT 시스템 프롬프트 정리
- 반복 작업용 템플릿 관리
