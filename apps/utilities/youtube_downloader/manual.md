# Video Downloader (영상 다운로드)

## 소개
YouTube, Vimeo, Twitter 등 다양한 웹사이트의 영상을 URL만으로 간편하게 다운로드합니다.

## 지원 사이트
- YouTube (동영상, 플레이리스트)
- Vimeo
- Twitter/X
- Instagram
- TikTok
- 기타 1000+ 사이트 (yt-dlp 지원)

## 사용법

1. 트레이/메뉴에서 **Video Downloader** 실행
2. 영상 URL 입력 (붙여넣기)
3. 옵션 설정:
   - **해상도**: 4K, 1080p, 720p, 480p 등
   - **포맷**: MP4, WebM, MP3 (오디오만)
   - **자막**: 자막 포함 여부
4. **Download** 버튼 클릭
5. 저장 폴더 확인

## 고급 기능
- **플레이리스트 다운로드**: 전체 목록 일괄 다운로드
- **오디오만 추출**: MP3로 음악만 저장
- **자막 다운로드**: 영상과 함께 자막 파일 저장
- **썸네일 저장**: 영상 썸네일 이미지 저장

## 저장 위치
- 기본: `Downloads/ContextUp/` 폴더
- Settings에서 변경 가능

## 의존성
- yt-dlp (영상 다운로드)
- FFmpeg (후처리)
