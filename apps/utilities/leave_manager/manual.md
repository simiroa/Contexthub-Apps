# Leave Manager (휴가 관리)

## 소개
연차/휴가 사용 내역을 시각적으로 관리하고 잔여 휴가일과 다음 휴가까지의 카운트다운을 표시합니다.

## 주요 기능
- **휴가 일정 관리**: 연차, 반차, 특별휴가 등록
- **잔여일 계산**: 총 연차 대비 사용량 자동 계산
- **카운트다운**: 다음 휴가까지 남은 일수 표시
- **달력 뷰**: 월별 휴가 현황 시각화
- **공휴일 표시**: 한국 공휴일 자동 표시

## 사용법

1. 트레이 메뉴에서 **Leave Manager** 실행
2. 연간 휴가 일수 설정
3. 휴가 일정 추가:
   - 날짜 선택
   - 휴가 유형 선택 (연차/반차/특별휴가)
   - 메모 입력 (선택)
4. 저장

## 기능
- **통계**: 월별/연도별 휴가 사용 현황
- **내보내기**: 휴가 목록 CSV 저장
- **알림**: 휴가 하루 전 알림 (선택)
