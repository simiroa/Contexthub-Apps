# Demucs Stem Separation (AI 음원 분리)

## 소개
Meta의 Demucs AI 모델을 사용하여 음악 파일에서 보컬, 드럼, 베이스, 기타 요소를 각각 별도의 트랙으로 분리합니다.

## 주요 기능
- **4-Stem 분리**: 보컬, 드럼, 베이스, 기타
- **고품질 분리**: 딥러닝 기반 정밀 소스 분리
- **배치 처리**: 여러 곡 일괄 처리 가능
- **무손실 출력**: WAV 포맷으로 저장

## 사용법

1. 음악 파일(MP3, WAV, FLAC 등) 선택 후 우클릭
2. **AI → Demucs Stem Separation** 선택
3. 처리 완료까지 대기 (곡당 2-5분)
4. 결과물 확인

## 출력 파일
분리된 각 트랙이 폴더에 저장됩니다:

```
곡명/
├── vocals.wav    # 보컬
├── drums.wav     # 드럼
├── bass.wav      # 베이스
└── other.wav     # 기타 악기
```

## 활용 예시
- 노래방 반주 제작 (보컬 제거)
- 드럼 연습용 트랙 추출
- 리믹스/매시업 제작
- BGM에서 멜로디 추출

## 시스템 요구사항
- NVIDIA GPU 권장 (CUDA)
- VRAM 4GB 이상
- CPU 모드도 지원 (느림)
