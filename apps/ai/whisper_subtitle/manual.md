# Whisper Subtitle AI (AI 자막 생성)

## 소개
OpenAI의 Whisper 음성인식 AI를 사용하여 영상/오디오의 음성을 텍스트로 변환하고 자막 파일을 자동 생성합니다.

## 주요 기능
- **다국어 지원**: 한국어, 영어, 일본어 등 99개 언어
- **자동 언어 감지**: 언어 미지정 시 자동 판별
- **타임스탬프 동기화**: 음성에 맞춘 정확한 자막 타이밍
- **다양한 모델**: tiny ~ large 모델 선택 가능

## 사용법

1. 영상/오디오 파일 선택 후 우클릭
2. **AI → Whisper Subtitle AI** 선택
3. 옵션 설정:
   - **언어**: 자동감지 / 한국어 / 영어 등
   - **모델 크기**: tiny, base, small, medium, large
   - **출력 형식**: SRT / VTT
4. **Generate** 버튼으로 생성

## 모델 크기 비교

| 모델 | VRAM | 속도 | 정확도 |
|------|------|-----|--------|
| tiny | ~1GB | 가장 빠름 | 낮음 |
| base | ~1GB | 빠름 | 중간 |
| small | ~2GB | 중간 | 중상 |
| medium | ~5GB | 느림 | 높음 |
| large | ~10GB | 매우 느림 | 최고 |

## 출력 파일
- `영상명.srt`: SubRip 자막 파일
- `영상명.vtt`: WebVTT 자막 파일

## 시스템 요구사항
- NVIDIA GPU 권장 (CUDA)
- 모델에 따라 VRAM 1~10GB
