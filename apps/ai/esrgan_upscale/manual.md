# ESRGAN Upscale (AI 업스케일)

## 소개
Real-ESRGAN 딥러닝 모델을 사용하여 저해상도 이미지의 디테일을 복원하면서 해상도를 최대 4배까지 높입니다.

## 주요 기능
- **4x 업스케일**: 512×512 → 2048×2048
- **디테일 복원**: 압축 노이즈 제거 및 선명도 향상
- **얼굴 향상**: GFPGAN 모델로 인물 이미지 품질 개선
- **배치 처리**: 여러 이미지 일괄 업스케일

## 사용법

1. 이미지 파일(들) 선택 후 우클릭
2. **AI → ESRGAN Upscale** 선택
3. 옵션 설정:
   - **배율**: 2x / 4x
   - **얼굴 향상**: 인물 사진인 경우 활성화
   - **디노이즈**: 노이즈 제거 강도
4. 처리 완료 시 `*_upscaled.png` 파일 생성

## 지원 포맷
- **입력**: PNG, JPG, JPEG, WebP, BMP
- **출력**: PNG (무손실)

## 시스템 요구사항
- NVIDIA GPU (CUDA 지원) 권장
- VRAM 4GB 이상 권장
- CPU 모드도 가능 (느림)

## 의존성
- PyTorch (CUDA)
- Real-ESRGAN
- GFPGAN (얼굴 향상)
